# Canvas Courseware VNext Plan

本文档用于记录下一版本的规划、范围边界和开发批次，不直接替代当前执行中的 `docs/TODO.md`。

## 版本定位

- 当前需求池中已进入版本计划的事项均已完成并转出 `docs/REQUIREMENTS.md`。
- 下一版本改为聚焦“质量收口与一致性修复”，优先关闭测试文档中仍未收口的阻断性问题。
- 当前质量收口问题最初共有 3 项，其中 `VNext-16` 页面背景独立性、`VNext-17` 文本双击编辑写回与 `VNext-18` 文本缩放链路都已完成验证。
- 在完成质量收口后，继续推进一组已在现有测试文档中显式出现的图片 / 背景体验增强功能。
- 下一版本继续遵守 UI 层与事件层解耦原则，不引入新的引擎耦合型实现。

## 暂不纳入下一版本

- 组合 / 打散
- 协同编辑
- 模板市场
- 图片 / PDF 导出
- 多引擎同时接入
- 新的功能型需求：待重新写入 `docs/REQUIREMENTS.md` 并完成评审后，再纳入后续版本规划

# Canvas Courseware VNext Plan

本文档用于记录下一版本的规划、范围边界和开发批次，不直接替代当前执行中的 `docs/TODO.md`。

## 版本定位

- 下一版本聚焦“创作效率与编排能力”，不再延续当前版本已进入修复中的工作台布局、滚动与信息精简问题。
- 版本目标是降低重复操作成本，补齐高频编辑能力，并让 slide 编排、图层管理、时间轴配置形成更完整闭环。
- 优先解决“做得慢、排版费劲、页面编排不顺、时间轴难维护、内容导入不方便”等核心体验问题。
- 下一版本的页面壳层与通用交互控件统一基于 Arco Design Vue 重构，建立一致的工作台 UI 语言与组件规范。

## 核心功能范围

- [x] Arco Design 页面重构
  说明：
  `apps/playground` 与 `packages/vue` 的页面壳层、导航区、面板区、表单区、弹窗区、抽屉区、标签与按钮等通用 UI，统一采用 Arco Design Vue 组件体系重构。
- [x] 撤销 / 重做
  说明：
  覆盖新增、删除、移动、缩放、属性修改、层级调整、slide 管理、timeline 步骤编辑等主要操作。
- [x] 快捷键体系
  说明：
  支持删除、撤销、重做、复制、粘贴、重复、方向键微调和大步移动。
- [x] 复制 / 粘贴 / 重复
  说明：
  支持同页复制、粘贴、重复节点，以及跨 slide 粘贴节点。
- [x] 多选与批量排版
  说明：
  支持框选、多选、批量移动、对齐与分布。
- [x] Slide 管理增强
  说明：
  支持复制、拖拽排序、删除确认、后插入与命名。
- [x] 图层管理增强
  说明：
  支持图层重命名、锁定 / 解锁、隐藏 / 显示，以及列表中定位选中节点。
- [x] 时间轴编排增强
  说明：
  支持步骤重命名、排序、复制、删除、从当前步骤开始预览、一键重置当前 slide 播放状态，并补齐 `duration` / `delay` 参数。
- [x] 本地图片导入
  说明：
  支持直接选择本地图片文件导入，并在项目持久化后恢复。
- [ ] 页面背景配置
  说明：
  支持按 slide 设置背景色、背景图和基础填充方式。
  当前推进范围：
  先完成按 slide 设置背景色、背景图与基础填充方式能力。
- [ ] 重新评估并修复 `BUG-05`
  说明：
  `Text` 对象缩放仍未生效，当前保持挂起，待后续重新评估。

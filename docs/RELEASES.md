# Canvas Courseware Releases

本文档用于归档已经完成的版本、阶段性成果和对应交付内容。每完成一个版本或一个明确阶段，应及时补充本文件。

## Release 01 - MVP 基线完成

### 版本目标

- 完成 Canvas Courseware SDK MVP 基础闭环。
- 打通项目列表、正式工作台、编辑态、预览态、本地持久化与 JSON 导入导出。
- 建立 UI 层与事件层解耦、文档模型与渲染引擎解耦的基础架构。

### 已完成范围

- 完成通用 schema、事件协议、store 与 controller 基础实现。
- 完成 `packages/core`、`packages/fabric`、`packages/vue` 与 `apps/playground` 基础骨架。
- 完成编辑态与播放态适配器。
- 完成项目列表页与正式工作台页。
- 完成 `Text / Image / Rect` 三类基础元素创建与编辑。
- 完成多 slide 管理、属性面板、层级列表与层级调整。
- 完成 timeline 基础配置 UI。
- 完成编辑 / 预览模式切换。
- 完成本地项目持久化与 JSON 导入导出。

### 关键修正

- 修复编辑态仍按节点初始 `visible` 状态隐藏后续步骤对象的问题，确保编辑态默认全显。
- 补齐编辑态步骤归属可视化。
- 修复工作台自适应布局问题，解决非预期滚动条与内容溢出。
- 重构工作台信息架构、三栏布局、滚动交互与默认信息密度。

### 已知说明

- 当前版本完成的是 MVP 基线与工作台可用性收口，不代表下一版本效率能力已全部完成。
- `BUG-05`：`Text` 对象缩放问题未在本阶段完成，后续继续评估。

## Release 02 - VNext 第一批完成

### 版本目标

- 完成下一版本第一批“编辑效率基础”能力。
- 建立统一的 Arco Design 页面壳层与通用交互语言。
- 补齐高频编辑能力，降低重复操作成本。

### 已完成范围

- 完成基于 Arco Design Vue 的页面壳层与通用 UI 重构。
- 完成项目列表页、工作台页、编辑壳层、预览壳层 Arco 化改造。
- 完成全局色彩、布局 token 与工作台视觉层级统一。
- 完成撤销 / 重做能力，并完成相关复测。
- 完成快捷键体系。
- 完成节点复制 / 粘贴 / 重复。
- 完成框选、多选、批量移动、对齐与分布。
- 完成 slide 管理增强：复制、拖拽排序、删除确认、后插入与命名。
- 完成图层管理增强：重命名、锁定 / 解锁、隐藏 / 显示、选中定位。

### 关键修正

- 修复撤销 / 重做按钮在变更后仍保持禁用的问题。
- 修复多选批量操作历史记录拆分问题，确保批量移动 / 对齐 / 分布支持单次撤销 / 重做。
- 多轮 UI 评审后持续压缩工作台顶部、侧栏和时间轴视觉层级，统一交互语言与组件风格。

### 未纳入本阶段

- 时间轴编排增强
- 本地图片导入
- 页面背景配置
- `BUG-05` 重新评估与修复

### 当前后续

- 下一步进入 VNext 第二批：编排与内容能力。
